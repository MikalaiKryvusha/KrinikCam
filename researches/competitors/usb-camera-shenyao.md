# «USB Camera» (ShenYao China) + «USB Camera Pro»

> Категория: USB/UVC-**вьювер/рекордер** (free), а **USB Camera Pro** умеет ещё и **стримить** —
> т.е. Pro частично прямой конкурент. Статус сбора: 🔬 сырьё (2026-06-29).

## Обзор

Самое популярное «USB Camera» на Android — от разработчика **沈垚 / ShenYao China**. Категория в сторе:
«Video players & editors». Превращает Android в видоискатель/рекордер для широкого спектра USB-видео:
вебки, грабберы, эндоскопы, тепловизоры, EasyCap. Есть платная **USB Camera Pro — «Capture, Stream &
Record»** (т.е. Pro добавляет стриминг). Package id (free): `com.shenyaocn.android.usbcamera`; Pro:
`com.shenyaocn.android.usbcamerapro`.

## Платформы и доступность

- **Android** (есть сборка для Android 11+, актуальные версии 10.x).
- Рейтинг free-версии: **2.84/5** из ~**16 000** оценок (низковато — много жалоб на совместимость).
- Огромная инсталл-база (одно из самых известных «USB camera» приложений).

## Цены / монетизация

- **USB Camera (free)** — бесплатно (реклама/ограничения).
- **USB Camera Pro** — платная (разовая покупка/паид), добавляет стриминг и расширенные функции.

## Ключевые фичи

- Просмотр и запись с **UVC-вебок, UVC-грабберов с аудио, EasyCap** (множество чипсетов: UTV007,
  HTV600/800, STK1160+SAA7113/GM7113, EM2860, SMI2021 и т.д.), **до 4K через HDMI-грабберы**.
- Поддержка **Windows Hello / IR-камер**, **USB-эндоскопов**, тепловизоров.
- **SBS (Side-by-Side)** для FPV-очков.
- Наложение **timestamp/GPS/скорость** на кадр; **loop-recording**, авто-сегментация, авто-удаление
  старых архивов при нехватке места → режим **видеорегистратора (dashcam)**.
- **Motion Detection** с авто-записью, **FTP-загрузкой** и email-уведомлениями.
- **USB Camera Pro**: дополнительно **стриминг** (RTMP/RTSP — уточнить точные протоколы) + запись.

## Поддержка USB/UVC

- Очень широкая совместимость (вебки/грабберы/эндоскопы/EasyCap/4K-HDMI). **Userspace-драйвер**,
  доступен только самому приложению (Android не даёт ставить kernel-драйвер для третьих).
- На **Android 9+** нужно разрешение **Camera** для полного доступа к USB-видео (как и у нас — Bug-2
  из нашей истории про CAMERA permission для UVC).

## Стриминг

- Free — нет стрима (просмотр/запись). **Pro — есть стрим** (вероятно RTMP/RTSP) → Pro = частично
  прямой конкурент. Глубина/UX стрима у утилитарного приложения, как правило, ниже.

## UX-заметки

Сугубо **утилитарный, технический** интерфейс (FPV/эндоскоп/dashcam/тепловизор аудитория). Не
«стримерский» продукт по подаче; много настроек, но не дружелюбно для блогера.

## Сильные стороны

- Беспрецедентно широкая аппаратная совместимость (включая экзотику: тепловизоры, эндоскопы, EasyCap).
- 4K через HDMI-грабберы, аудио с грабберов.
- Много утилитарных режимов (dashcam, motion detection, FTP).
- Pro умеет стримить.

## Слабые стороны и БОЛИ пользователей (для позиционирования)

- **Низкий рейтинг 2.84/5** — массовые жалобы на **совместимость/нестабильность** с конкретными
  камерами/устройствами (типично для userspace-UVC).
- **Утилитарный, не стримерский UX** — не заточен под «вышел в эфир на YouTube».
- Реклама во free; стрим только в Pro.
- _(Свежие конкретные цитаты из Play Store/Reddit добрать.)_

## Релевантность для KrinikCam (без оскорблений)

- Это «швейцарский нож» для USB-видео (грабберы/эндоскопы/dashcam) — но **не стримерский** продукт.
  KrinikCam узко-заточен под сценарий стримера: вебка → красивый fullscreen → эфир, с понятным UX.
- Наши плюсы: **современный стримерский UX**, профили платформ, standby-кадр, open-source, фокус.
- Где они сильнее: широта железной совместимости (включая экзотику) и утилитарные режимы — нам не
  обязательно туда лезть (другая аудитория), но совместимость камер — важный урок (тестировать больше
  моделей UVC).
