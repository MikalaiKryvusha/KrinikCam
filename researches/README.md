# `researches/` — база знаний по большим трудным вопросам

Конспекты и выжимки по масштабным сложным вопросам: исследования, накопленное ноу-хау, важная
справочная информация, которую было трудно собрать и которая не должна испариться. Здесь живут:
анализ конкурентов (`competitors/`), знания по GL-композитору, SEO-черновики и т.п.

**Кринику (владельцу):** место, где найти глубокий бэкграунд решений — почему выбран подход, что
выяснили о трудной проблеме. Можешь обозначить тему, которую хочешь исследовать.

**ИИ-агенту:** если вопрос настолько велик, что его результаты должны пережить текущую задачу — пиши
конспект сюда (сырые источники → анализ → выводы/гипотезы). Ссылайся на него из бага/плана/идеи,
которые его породили (DRY — не исследуй заново). Конспект — **живой справочник**, а не закрываемая
задача: тег `DONE` не ставится, обновляй по мере роста понимания.
