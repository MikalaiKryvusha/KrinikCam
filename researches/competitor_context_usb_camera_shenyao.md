- Есть автор-программист с его авторсикм сайтом, посвящённым его проектам: https://www.shenyaocn.com/


- У него есть мобильное Android приложение USB камеры - оно позволяет с подклёчённой по USB камере к Андроид устройсту:
 - записывать видео с камеры в файл на устройсе
 - стримить в локальную сеть с камеры
 - стримить в YouTube и Instagram прямые трансляции.

Вот описание этого приложения от самого автора:
Поддерживает следующие устройства:

* UVC веб-камера с микрофоном или внешний USB аудиовход (поддерживает H.264, H.265, HEVC, MJPG, YUY2, P010, NV12 и другие форматы)

* UVC видеозахват с аудиовходом или внешний USB аудиовход (до 4К через HDMI, поддерживает Н.264, H.265, HEVC, MJPG, YUY2, P010, NV12 и другие форматы, поддерживает прогрессивное и чередующееся видео)

Камера Windows Hello и инфракрасная камера

* Geek szitman supercamera (2се3:3828) — устройства без UVC, например, USB эндоскоп

* EasyСар, включая аудио с чипами UTV007/HTV600/HTV800

* EasyСар, включая аудио с чипами STK1160 + SAA7113/GM7113 + AC97 (стерео версия 48kHz)

* EasyСар, включая аудио с чипами STK1160 + SAA7113/GM7113 (моно версия 8kHz)

* EasyСар, включая аудио с чипами ЕМ2860 + SAA7113/GM7113 + AC97

* EasyСар, включая аудио с чипами SMI2021 + SAA7113/GM7113 + ES7240/CS5340

Пожалуйста, нажмите на иконку USB на панели инструментов, чтобы открыть диалоговое окно устройства, если система не может обнаружить ваше устройство.

Пожалуйста, переключите видео-стандарт (PAL/NTSC/SECAМ), если видео зависает при использовании EasyCap.

Пожалуйста, используйте качественный OTG-кабель и обеспечьте надежное питание! Некоторые устройства могут потребовать использования хаба для правильной работы, например, Elgato Cam Link.

USB Камера позволяет вашему устройству на Android подключаться к

USB-камерам или видеозахватам через USB-OTG. Вы можете записывать видео или делать снимки, ИЛИ превратить ваш телефон в беспроводную IP-камеру через встроенные серверы RTSP и НТТР для мониторинга безопасности с поддержкой двустороннего аудио. Вы можете использовать свой браузер для просмотра, включая приложение IP Камера.

USB Камера может передавать видео и аудио на серверы RTMP/SRT в режиме прямой трансляции (например, на YouTube, Facebook), и использоваться для сетевых трансляций. Поддерживает протокол безопасности rtmps и протокол SRT, а также может передавать медиа на несколько серверов одновременно. Также поддерживает HEVC/AV1 через RTMP и может использоваться для YouTube Live.

Она поддерживает автоматическую запись видео на основе детекции движения, и видео можно автоматически загрузить на FTP-сервер и уведомить вас по электронной почте!

Поддерживает циклическую запись. Можно настроить автосегментацию при записи и автозачистку старых видеозаписей при недостаточности места для хранения. USB Камера может использоваться как Dash Cam.

USB Камера поддерживает плавный переход между передним и фоновым режимом. Просто нажмите 'Войти в фоновый режим в меню. Запись не будет прервана во время переключения!
