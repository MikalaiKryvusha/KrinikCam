Цель:
- Разработать удоброе, быстрое, функциональное, современное, гибкое Андроит 64 bit приложение для захвата видео и параллельного стриминга с внешних Веб-камер, подключенных к Android устройству, в различные интернте платформы.

Философия:
 - Открытость - проект OpenSource. Доступен всем и каждому на странице Github
/Users/kryvusha/ai_sandbox/KRINIKS_ANDROID_USB_WEB_CAMERA_FOR_STREAMING/plans/github.md

 - "УМНОСТЬ" - приложение само, без обязательного вмешательства юзера, понимает, на каком устройсве его запустили, соответствующим образом включая тот или иной профиль работы - профили переключаются легко и просто, каждый профиль - набор предзаготовленных настроек (configs) для того, чтобы наилучим образом работать именно на устройсве юзера.
