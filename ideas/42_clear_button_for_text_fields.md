# Idea 42 — Кнопка-крестик мгновенной очистки для полей URL / ключа

> Автор: Криник (чат, 2026-07-19). Статус: ПРИНЯТА (видение Криника), ждёт реализации.

## Суть

У полей ввода, требующих длинных значений (RTMP URL, ключ трансляции), добавить **кнопку-крестик (✕)**
внутри поля справа — мгновенная очистка всего поля одним тапом (не стирать посимвольно).

- Показывать крестик, когда поле НЕ пустое (стандартный паттерн `trailingIcon` в `TextField`).
- Тап → поле очищается, фокус остаётся (удобно вставить новое значение).
- Применить к: RTMP URL и Stream Key (форма платформы, ProfileEditDialog), и другим длинным текст-полям.

## Зачем

Ключи/URL длинные, вставляются/меняются часто; стирать вручную — раздражает. Крестик — привычный, быстрый
способ очистить поле перед новой вставкой.

## Опора

- Compose `OutlinedTextField`/`TextField` имеет `trailingIcon` — туда `IconButton(Icons.Default.Clear)`,
  видимый при `value.isNotEmpty()`. Единый мелкий helper-компонент, переиспользовать во всех таких полях.
