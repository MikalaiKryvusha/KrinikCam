# `ideas/` — идеи и предложения по фичам

Проработанные идеи, ЧТО строить — обычно узкий срез проекта, описанный достаточно, чтобы агент мог
реализовать. Чаще всего пишет **Криник**, но агент тоже предлагает идеи (навык `/propose-idea`). Одна
идея — один файл `NN_<name>.md`.

**Кринику (владельцу):** это твоя главная директория. Кидай сюда идею своими словами — агент причешет
её в структурный вид и реализует. Идея — элемент видения продукта: агент реализует её только после
твоего одобрения.

**ИИ-агенту:** идеи Криника читай, правь опечатки, минимально структурируй — и реализуй. Своя стоящая
идея — оформи здесь со статусом «❓ НА СОГЛАСОВАНИИ КРИНИКА» (навык `/propose-idea`) и НЕ реализуй до
одобрения. После реализации впиши в файл статус и дату и проставь тег DONE
(`git mv NN_x.md NN_DONE_x.md`).
