# /what-next — предложить самые ценные следующие шаги

Иногда агент останавливается и ничего не предлагает. Этот навык — ответ одной командой на простой
вопрос владельца — **«что дальше?»** — план следующих шагов, ранжированный по ценности для видения.

Все пути относительно корня проекта: `/Users/kryvusha/ai_sandbox/KrinikCam`.

## Процедура

### Шаг 1. Сориентироваться (дёшево, без глубокого погружения)
Прочитай `GOAL.md` (видение), `MASTER_PLAN.md` (в какой фазе мы), `STATUS.md` (где мы сейчас) и
пройдись по открытому беклогу: `bugs/`, `ideas/`, `plans/`, `homeworks/` без тега `DONE`, плюс
неотвеченные `interviews/`.

### Шаг 2. Ранжировать по ценности
Упорядочи шаги-кандидаты по **ценности для видения** согласно `PHILOSOPHY.md`: Парето (те немногие,
что двигают результат), матрица Эйзенхауэра (важное × срочное), эффекты второго порядка (что
разблокирует больше всего будущей работы). Отметь примерную трудоёмкость каждого.

### Шаг 3. Ответить в чат
1. **ОДИН следующий шаг** — самый ценный, и *почему именно он* (привяжи к GOAL/MASTER_PLAN).
2. **2–4 запасных** — по строке на каждый, с ценностью/трудоёмкостью.
3. **Заблокировано на Кринике** — открытые интервью/домашки, если есть.

### Шаг 4. Предложить начать
Предложи немедленно начать верхний шаг; по подтверждению Криника (или в автономном цикле) — начинай.

## Правила
- Никогда не отвечай «делать нечего»: пустой беклог = предложи `/check-backlog` или
  `/refresh-context`, работу над качеством/техдолгом, или `/interview` для пополнения видения.
- Будь конкретен: шаги с именами и файлами, а не общие слова.
