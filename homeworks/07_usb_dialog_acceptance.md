# ДЗ 07 — Приёмка фикса USB-диалога (bug 33), ~3 минуты

> Заведено: 2026-07-18, ночной цикл. Фикс стоит на планшете (debug-билд 0.7(4)).
> Суть фикса: не-камеры больше НЕ вызывают диалог вообще; камера — максимум ОДИН диалог на устройство.

## Что сделать (по шагам, приложение KrinikCam открыто)

1. **Не-камера:** воткни в OTG что-нибудь НЕ-камерное (мышь / флешка / клавиатура).
   **Ожидание:** НИКАКОГО диалога от KrinikCam. Устройство в списке источников НЕ появляется.
2. **Камера, первое втыкание:** воткни вебку (Piko+).
   **Ожидание:** системный диалог появится ОДИН раз →
   ⚠️ **ПОСТАВЬ ГАЛОЧКУ «Всегда открывать KrinikCam…» / «По умолчанию»** — это единственный
   механизм Android, чтобы грант запомнился навсегда. Потом «ОК».
3. **Переподключение:** выдерни вебку → воткни снова (можно несколько раз, можно через зарядку).
   **Ожидание:** диалога БОЛЬШЕ НЕТ, камера подхватывается сама.
