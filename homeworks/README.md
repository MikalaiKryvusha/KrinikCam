# `homeworks/` — задания от агента Кринику

Задачи, которые агент просит сделать **человека** — то, что сам не может из-за своей цифровой
бестелесной природы: проверить на живом железе (камера/YouTube/руки), действовать в физическом мире,
использовать аккаунт/креды, которые есть только у Криника. Каждый документ описывает задачу
конкретными шагами и собирает обратно наблюдения и результаты Криника. Один файл — `NN_<name>.md`.

**Кринику (владельцу):** если агент завёл ДЗ — ему нужна рука в физическом/офлайн мире. Пройди шаги и
впиши, что наблюдал, прямо в документ — агент прочитает и продолжит. Девайс-логи клади в
`homeworks/logs/` (в git не коммитятся), скриншоты — в `tools/screenshots/`.

**ИИ-агенту:** упёрся в то, что может сделать только человек-с-телом — не стой: напиши сюда ДЗ с
ясными, минимальными, нумерованными шагами и местом для результатов, и продолжай другую работу. Когда
Криник отчитается — учти результаты и проставь тег DONE (`git mv NN_x.md NN_DONE_x.md`).
