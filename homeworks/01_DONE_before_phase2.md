## 2. Разрешение на USB — диалог

Сбрось разрешения приложению: **Настройки → Приложения → KrinikCam → Разрешения → Камера → Запретить**.
Затем заново открой приложение с подключённой камерой.

**Что проверить:**
- [ ] Появился ли диалог запроса разрешения камеры от Android?
- [ ] После выдачи — появился ли диалог "Разрешить KrinikCam доступ к USB-устройству"?
- [ ] После обоих "Разрешить" — показывает ли превью без перезапуска?

**Твои наблюдения:**
> Сбросил хранилище приложения. Запустил -  приложение спросило разрешения на камеру и микрофон - выдал - запустилось видео.
Если в настройках приложения ОС запретить разрешения - то больше оно их не спрашивает - нужно бы добавить опцию в настройках ВЫЗВАТЬ разрешения ещё раз прямо из настроек, чтобы не лазить в настройки ОС.
> Да, этот диалог "Разрешить KrinikCam доступ к USB-устройству"? появился.
> Да, сразу запускается видео с камеры.

## 5. Тест RTMP стрима (опционально, если есть ключ)

Если у тебя есть stream key от YouTube или Twitch — попробуй запустить стрим.

**Для теста без реального стрима:** можно использовать бесплатный тест-сервер:
- RTMP URL: `rtmp://live.restream.io/live`

**Что проверить:**

- [ ] Если стрим пошёл — видно ли превью, есть ли индикатор что идёт стриминг?
- [ ] Кнопка остановки стрима работает?

**Твои наблюдения:**
> Кнопка [Go Live] нажимается, но приложение скрешилось. Я попробовал дважды я сключенным профилем моего YouTube, и один раз с выключенным профилем.
