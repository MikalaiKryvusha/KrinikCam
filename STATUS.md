## Ключевые константы проекта

- **Устройство:** Headwolf Titan 1 (serial `DHF8256GB25442874`, Android 16/SDK 36), ADB WiFi — IP
  динамический (переподнятие через USB: `adb tcpip 5555` → `adb connect <ip>:5555`).
- **Камера-тест:** Emeet Piko+ 4K (сейчас НЕ подключена — работаем на виртуалке).
- **Сборка:** ТОЛЬКО JBR (`JAVA_HOME="/Applications/Android Studio.app/Contents/jbr/Contents/Home"`)
  + `node tools/build.mjs --no-ui`.
- **Git:** только `main`, коммит+пуш = `node tools/commit.mjs "..."`.
- Решения Криника по продукту — `MASTER_PLAN.md` §10 и архив (`researches/status_archive.md`, секция
  «Важные решения»).
