# `interviews/` — решения уровня владельца

Интервью, которые агент проводит с Криником, чтобы закрыть решения, которые агент НЕ вправе принимать
сам: UI/UX, серьёзные технические развилки, бренд/видение/приоритеты. Закрытые вопросы A/B/C/D с
рекомендацией на первом месте; Криник отвечает **прямо в документе**. Один файл —
`interview_NNN_<topic>.md`.

**Кринику (владельцу):** если агент завёл интервью — он ждёт **тебя**. Заполни поля «**Ответ:**» прямо
в документе (выбери A/B/C или впиши своё в D). Здесь фиксируются и сохраняются твои судьбоносные
решения.

**ИИ-агенту:** заводи интервью только для действительно владельческих развилок (навык `/interview`).
Варианты — **A/B/C/D**: **A** — всегда выбор, отфильтрованный через `PHILOSOPHY.md` (простейший и
самый эффективный), с пометкой **(рекомендую)**; **D** — всегда «свой ответ» владельца. Сначала сделай
подготовительную работу, вопросов 1–5, затем пауза — жди ответов Криника. Всё дёшево-обратимое решай
сам, без интервью.
