# Bug 33 — Навязчивый системный USB-диалог «Разрешить доступ к камере» на КАЖДОМ переподключении

> Заведён: 2026-07-07 (Криник, дома, live). Приоритет: ВЫСОКИЙ (бесит пользователя ежедневно).
> Связано: plans/04 §Фикс 29.1 (S1 не сделан), bug 09 (манифест-автогрант).

## Симптом (со слов Криника)

«Когда отключаю зарядку, подключаю камеру — этот диалог "Можно ли использовать камеру" ПОСТОЯННО
появляется». Договаривались, что он не будет навязчиво показываться — а он лезет на каждом
переподключении USB-камеры (отключил зарядку → воткнул камеру через OTG → снова диалог).

## Причина (гипотеза — plans/04 §29.1 S1, НЕ реализован)

`AUSBC/USBMonitor` при `DeviceAttached` зовёт системный `requestPermission` БЕЗ предварительной
проверки `usbManager.hasPermission(device)`. Грант разрешения на устройство (VID/PID) персистентен,
но код всё равно каждый раз просит → системный диалог каждый раз. S2 (дебаунс дубликатов в окне 3с,
2026-07-06) убрал СТОПКУ одинаковых, но не убрал ОДИН диалог на каждое подключение.

Манифест-автогрант (bug 09, `device_filter.xml` class 239/2) покрывает запуск приложения при уже
воткнутой камере, но НЕ hot-plug при уже открытом приложении.

## План фикса (plans/04 §29.1 S1)

- [ ] Перед `requestPermission` проверять `usbManager.hasPermission(device)`:
      - если TRUE → сразу открывать камеру, НЕ звать `requestPermission` (нет диалога);
      - если FALSE → запросить один раз (грант персистентный, «всегда разрешать» для устройства).
- [ ] Проверить, что грант реально ПЕРСИСТИТ (пользователь мог не отмечать «всегда»; на некоторых ОС
      грант живёт до отключения устройства — тогда hot-plug всё равно спросит один раз). Изучить, можно
      ли шире покрыть `device_filter.xml` / intent-filter USB_DEVICE_ATTACHED (авто-грант при attach).
- Место: `UsbDeviceRepositoryImpl.requestPermission` / `UsbViewModel` (перед запросом).

## Как воспроизвести

1. Приложение открыто. Отключить зарядку, воткнуть USB-камеру через OTG (hot-plug).
2. Наблюдать системный диалог «Разрешить … доступ к USB-устройству?» — и так каждый раз.
