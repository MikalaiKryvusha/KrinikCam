# Bug 32 — Встроенная СЕЛФИ-камера растянута (искажён аспект)

> Заведён: 2026-07-07 (найтлуп, при тесте S4 выбора источника). Криник подтвердил визуально.
> Устройство: Titan 1. Сборка `com.kriniks.kcam.debug` 0.5(50).

## Симптом
При выборе источника «Селфи-камера (id 1)» (встроенная фронтальная, Camera2) видео на превью
**растянуто** — неверный аспект (лицо шире/выше натурального). У UVC-вебки такого нет.

## Гипотезы (проверить)
1. **Аспект сенсора ≠ 16:9.** Фронтальная камера отдаёт, вероятно, 4:3 (или иной), а мы кормим её в
   фикс. 16:9-буфер FBO без аспект-коррекции/кропа → растяг по одной оси. Двухпроходный FBO-рефактор
   (plans/04) чинил аспект для UVC/2K; для встроенных Camera2, возможно, не выбран корректный
   16:9-размер выхода ИЛИ texMatrix Camera2 не учитывает соотношение.
2. **DeviceCameraOpener** запрашивает размер, не соответствующий 16:9 → SurfaceTexture буфер не 16:9 →
   при рендере квадом растягивается. Нужно выбирать ближайший поддерживаемый 16:9 output size из
   `StreamConfigurationMap` (аналог bug 25 для UVC) ИЛИ применять центр-кроп.
3. Возможна связь с ориентацией сенсора фронталки (SENSOR_ORIENTATION), но симптом = именно РАСТЯГ,
   не поворот, так что первично аспект.

## Как воспроизвести
1. Открыть панель «Слои» → слой «Устройство захвата видео» → «Источник» → «Селфи-камера».
2. Превью фронталки растянуто по сравнению с натуральным.

## Заметки к фиксу
- Смотреть `DeviceCameraOpener` (app/streaming/DeviceCamera.kt): какой preview/output size выбирается
  для Camera2 и как настраивается `SurfaceTexture.setDefaultBufferSize`. Выбрать 16:9 output из
  поддерживаемых (или ближайший + центр-кроп в композиторе через texMatrix).
- Сверить с тем, как UVC-путь выбирает размер (там аспект ок) и как композитор рендерит камеру-квад.
- Тыловая «Основная камера (id 0)» — проверить, есть ли тот же растяг (аспект тыла может отличаться).

## Статус
🔴 ОТКРЫТ. Приоритет: средний (встроенные камеры — валидный источник, но UVC — основной для Криника).
Смежное: bug 29.2 (искажение device-камеры, закрыто FBO для UVC), bug 25 (выбор размера UVC).
